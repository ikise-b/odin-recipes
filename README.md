# Odin Recipes

A simple recipe website built with HTML as part of The Odin Project curriculum.

## Description

This project is a basic recipe website that showcases fundamental HTML skills. It features a homepage with links to individual recipe pages, demonstrating the use of HTML structure, lists, links, and images.

## Technologies Used

- **HTML5** - Structure and content
- **Git** - version control
- **VS Code** - Code editor
- **Bash/Linux Terminal** - Navigation and file/directory creation using command line. 

## Skills Demonstrated

- Creating HTML boilerplate
- Working with headings, paragraphs ans lists
- Adding links and images
- Using proper file structure and organization
- Basic Git workflow (add, commit, push)
- Command line navigation and file creation

## Project structure

odin-recipes/
```
├── assets                    # Images and other resources
├── index.html                # Homepage with recipe links
├── README.md
└── recipes                   # Individual recipe pages
```
